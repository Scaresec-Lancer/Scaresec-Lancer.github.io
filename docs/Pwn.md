# 关于Pwn

Pwn是CTF比赛中二进制方向的简称，著名赛事有：Pwn2Own、GeekPwn。

Pwn的基础学习需要Linux、汇编、C、C++、Python的基础，工具需要IDA、GDB等。

学习的时候需要熟练掌握C语言的函数调用栈、PLT、GOT表、ROP、heap利用等一些基础性的知识。

最后，Pwn的学习没有太多的捷径，多看代码多分析多刷题。
